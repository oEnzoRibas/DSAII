## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/oEnzoRibas/DSAII.git
```
2. Navigate to the folder of the algorithm you want to run:
```bash
cd src\main\java/
```

3. Compile and run the Java program:
```bash
javac BinaryTree.java
java BinaryTree
```
